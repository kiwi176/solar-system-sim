🌌 Solar System Simulator

An interactive solar system simulation built using JavaScript and p5.js.

✨ Features
🪐 Realistic planet orbits based on orbital periods
🌍 Detailed Earth with continents and clouds
🔍 Click on planets to zoom in
🎮 Smooth camera movement and zoom
⚡ Adjustable simulation speed using scroll
🕹️ Controls
Click on a planet → Zoom in
Click again → Zoom out
Scroll → Change simulation speed
🛠️ Built With
JavaScript
p5.js
